# 游戏模块
## 模块划分
游戏模块划分为common和game。common非特定于游戏，而是所有游戏都能使用的功能。game则通常具有唯一性，会经常被改变。

# 游戏模块
## [技能](file://./mod_game_skill.h)
	分为战斗技能和特殊技能。战斗技能如：神技、绝技、剑法等。特殊技能如：偷窃、口才、解锁等。
## [派别](file://./mod_game_camp.h)
	不同派别具有不同技能或不同技能、天赋优越成长线。在很多西方游戏中，也称为种族。
## [天赋]((file://./mod_game_.h)
	属性、技能成长优越。
## [装备]((file://./mod_game_.h)
	提升属性或增加技能。
	分为武器、防具、提升属性道具、增加技能道具等。
## [道具]((file://./mod_game_.h)
