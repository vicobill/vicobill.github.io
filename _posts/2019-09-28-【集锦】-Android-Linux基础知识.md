## 常见命令

常见Linux命令：

|命令|意义|完整英文|
|---|---|---|
| ls | 列出目录 | list |
| cd | 切换目录 | change directory |
| pwd | 显示当前目录 | print current directory |
| cp | 复制文件或目录 | copy |
| rm | 移除文件或目录 | remove |
| mv | 移动文件或目录 | move | 
| mkdir | 创建新的目录 | make directory |
| rmdir | 删除一个空的目录 | remove directory |
